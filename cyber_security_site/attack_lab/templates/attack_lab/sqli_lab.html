📚 Как работает SQL Injection?

Пользователь вводит данные → они вставляются в SQL → база выполняет запрос → злоумышленник может изменить логику. 

SELECT * FROM users WHERE username = 'input'

💥 Пример атаки

' OR 1=1 --

Условие всегда TRUE → возвращаются все пользователи.

🧠 Типы SQLi

  • Auth bypass
  • UNION-based
  • Error-based
  • Blind SQLi

🛡️ Защита

  • Параметризованные запросы
  • Django ORM
  • Валидация ввода
