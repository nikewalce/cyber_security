📚 Stored XSS (справочник + практика)

💥 Как работает Stored XSS?

Пользователь вводит данные → сервер сохраняет их в базу → другой пользователь открывает страницу → браузер выполняет код. 

User → input → Database → Page → Browser executes JS

🔥 Пример атаки 

<script>alert("Hacked")</script>

Этот код сохранится и выполнится у всех пользователей.

🍪 Cookie и XSS 

<script>\nalert(document.cookie)\n</script>

Если cookie НЕ защищены (нет HttpOnly), их можно украсть через XSS.

🛡️ Защита

  • Django autoescape
  • Form validation
  • escape()
  • CSP
  • HttpOnly cookies

🧰 Режимы

  • ❌ Vulnerable → XSS работает
  • ⚠️ Secure → через формы
  • 🛡️ Super Secure → escape()

🎯 Попробуй

  • <script>alert(1)</script>
  • <img src=x onerror=alert(1)>
