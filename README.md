# Cyber Security Learning Project

## 📚 О проекте

Этот репозиторий — мой учебный проект, в котором я изучаю **кибербезопасность, offensive security и разработку безопасных веб‑приложений**.

В проекте я:

* изучаю и реализую различные **техники атак**
* пишу собственные **инструменты для анализа сети**
* создаю **веб‑сайт на Django**, который позже буду использовать для тестирования уязвимостей

Основная цель проекта — **практическое изучение кибербезопасности через код**.

---

# 🎯 Цели проекта

* Изучить основы **offensive security**
* Реализовать популярные инструменты из области pentesting
* Научиться **искать и понимать уязвимости**
* Создать **собственную тестовую платформу на Django**
* Практиковаться в анализе сетевого трафика и логов

---

# 🧰 Реализованные инструменты

## 🔎 Port Scanner

Сканер портов, написанный на Python.

Функции:

* сканирование TCP‑портов
* определение открытых портов
* работа с сокетами
* понимание процесса TCP‑соединения

Используется для изучения:

* сетевых сервисов
* принципов работы TCP/IP
* методов разведки в pentest.

---

## 🐚 Reverse Shell

Простая реализация reverse shell на Python.

Используется для понимания:

* удалённого выполнения команд
* принципов работы shell‑соединений
* механизмов управления удалённой системой

⚠️ Используется **только в учебных целях**.

---

# 🌐 Django веб‑сайт

В проекте также создаётся веб‑сайт на Django, который будет использоваться как **платформа для изучения веб‑безопасности**.

Планируемые модули:

* 📰 Новости кибербезопасности

# 🚀 Запуск проекта

### 1. Клонировать репозиторий

```
git clone https://github.com/nikewalce/cyber_security.git
cd cyber_security
```

### 2. Создать виртуальное окружение

```
python -m venv .venv
```

Активировать:

Windows

```
.venv\Scripts\activate
```

Linux / Mac

```
source .venv/bin/activate
```

---

### 3. Установить зависимости

```
pip install -r requirements.txt
```

или

```
poetry install
```

---

### 4. Применить миграции

```
python manage.py migrate
```

---

### 5. Запустить сервер

```
python manage.py runserver
```

Сайт будет доступен по адресу:

```
http://127.0.0.1:8000
```
